## Getting started

The NoticeChoice component is to be utilized within IBM.com or IBM Authorized partner websites and is required on all registration forms.  Please note that this service displays the proper notice and choice content and uses country and user information to determine whether to pre-check the boxes.  It does NOT save the results of the userâ€™s selection.  You must use [MRS API](https://w3.ibm.com/w3publisher/mrs/mrsaas-api) to store the interaction along with the users choices.


Here's a quick example to get you started.

```css
// yourapplication.scss
```
