## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/feature-card/index.js';
```

### JS (via CDN)

```html
<!-- LATEST tag -->
<script type="module" src="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/latest/feature-card.min.js"></script>

<!-- NEXT tag -->
<script type="module" src="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/next/feature-card.min.js"></script>

<!-- BETA tag -->
<script type="module" src="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/beta/feature-card.min.js"></script>

<!-- SPECIFIC VERSION (available starting v1.6.0) -->
<script type="module" src="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/v1.6.0/feature-card.min.js"></script>
```
