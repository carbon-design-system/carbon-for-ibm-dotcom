## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/callout-quote/index.js';
```

<Description markdown={`${cdnJs({ components: ['callout-quote'] })}`} />

<Description markdown={`${cdnCss()}`} />

### SCSS

Callout quote has specific tokens per theme that need to emitted in the styles.
You can do this for example by adding the following:

```css
@use '@carbon/ibmdotcom-styles//scss/globals/tokens/color-tokens' as
  color-tokens;
@include add-component-tokens(color-tokens.$color-tokens);
```

### HTML

```html
<c4d-callout-quote mark-type="double-curved">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus est purus,
  posuere at est vitae, ornare rhoncus sem. Suspendisse vitae tellus fermentum,

    Lorem ipsum dolor sit amet
  </c4d-quote-source-heading>

</c4d-callout-quote>
```
