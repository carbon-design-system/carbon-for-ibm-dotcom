## Getting started

Here's a quick example to get you started.

##### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/content-group/content-group-heading.js';
import '@carbon/ibmdotcom-web-components/es/components/link-list/link-list.js';
import '@carbon/ibmdotcom-web-components/es/components/content-group-banner/content-group-banner.js';
import '@carbon-web-components/es/icons/arrow--right/20';
```

### HTML

```html
<dds-content-group-banner>
  <dds-content-group-heading>Curabitur malesuada varius mi eu posuere</dds-content-group-heading>
  <dds-card-cta slot="footer" href="https://www.ibm.com/" type="local">
    Lorem ipsum dolor sit amet
    <dds-card-cta-footer>
      ${ArrowRight20({ slot: 'icon' })}
    </dds-card-cta-footer>
  </dds-card-cta>
</dds-content-group-banner>
```
