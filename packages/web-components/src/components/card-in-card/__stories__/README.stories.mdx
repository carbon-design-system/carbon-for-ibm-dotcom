To render the video variation, make sure to wrap `card-in-card` with `video-cta-container`. However, if parent component (ie. `Card group`)
already is wrapped in `video-cta-container`, there is no need to wrap the `card-in-card`.

NOTE: The default video title can be overwritten by passing in the `video-name` attribute to `card-in-card`.

```html
<dds-video-cta-container>
  <dds-card-in-card video-name="custom card title" href="1_9h94wo6b" cta-type="video">
    <dds-card-eyebrow>${eyebrow}</dds-card-eyebrow>
    <dds-card-cta-footer cta-type="video" href="1_9h94wo6b"></dds-card-cta-footer>
  </dds-card-in-card>
</dds-video-cta-container>
```
