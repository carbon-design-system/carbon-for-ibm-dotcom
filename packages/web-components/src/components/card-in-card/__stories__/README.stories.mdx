To render the video variation, make sure to wrap `card-in-card` with
`video-cta-container`. However, if parent component (ie. `Card group`) already
is wrapped in `video-cta-container`, there is no need to wrap the
`card-in-card`.

NOTE: The default video title can be overwritten by passing in the `video-name`
attribute to `card-in-card`.

```html
<c4d-video-cta-container>
  <c4d-card-in-card
    video-name="custom card title"
    href="0_ibuqxqbe"
    cta-type="video">
    <c4d-card-eyebrow>${eyebrow}</c4d-card-eyebrow>
    <c4d-card-footer
      cta-type="video"
      href="0_ibuqxqbe"></c4d-card-footer>
  </c4d-card-in-card>
</c4d-video-cta-container>
```
