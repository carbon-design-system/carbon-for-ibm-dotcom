## Getting started

Here's a quick example to get you started.

## Callout with Media (Image)

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/callout-with-media/index.js';
import '@carbon/ibmdotcom-web-components/es/components/callout-with-media/callout-with-media-image.js';
```

<Description
  markdown={`${cdnJs({
    components: ['callout-with-media', 'callout-with-media-image'],
  })}`}
/>

<Description markdown={`${cdnCss()}`} />

### SCSS

Callout quote has specific tokens per theme that need to emitted in the styles.
You can do this for example by adding the following:

```css
@use '@carbon/ibmdotcom-styles//scss/globals/tokens/color-tokens' as
  color-tokens;
@include add-component-tokens(color-tokens.$color-tokens);
```

### HTML

```html
<c4d-callout-with-media>
  <c4d-content-block-heading
    >lorum ipsum dolor sit ameet</c4d-content-block-heading
  >
  <c4d-callout-with-media-copy size="sm"
    >lorum ipsum dolor sit ameet</c4d-callout-with-media-copy
  >
  <c4d-callout-with-media-image
    alt="Image alt text"
    default-src="http://fpoimg.com/672x378?text=16:9&bg_color=ee5396&text_color=161616"
    heading="Lorem ipsum dolor sit amet, consectetur adipiscing elit.">
  </c4d-callout-with-media-image>
</c4d-callout-with-media>
```
