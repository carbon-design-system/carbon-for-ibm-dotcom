## Getting started

Here's a quick example to get you started.

## Callout with Media (Image)

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/callout-with-media/index.js';
import '@carbon/ibmdotcom-web-components/es/components/callout-with-media/callout-with-media-image.js';
```

<Description markdown={`${cdnJs({ components: ['callout-with-media', 'callout-with-media-image'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<dds-callout-with-media>
  <dds-content-block-heading>lorum ipsum dolor sit ameet</dds-content-block-heading>
  <dds-callout-with-media-copy size="sm">lorum ipsum dolor sit ameet</dds-callout-with-media-copy>
  <dds-callout-with-media-image
    alt="Image alt text"
    default-src="http://fpoimg.com/672x378?text=16:9&bg_color=ee5396&text_color=161616"
    heading="Lorem ipsum dolor sit amet, consectetur adipiscing elit."
  >
  </dds-callout-with-media-image>
</dds-callout-with-media>
```
