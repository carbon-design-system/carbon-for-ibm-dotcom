### HTML

```html
<dds-content-section>
  <dds-content-section-heading>Heading</dds-content-section-heading>
  <dds-content-section-copy>Copy</dds-content-section-copy>
  <dds-text-cta slot="footer" cta-type="local" href="https://www.example.com">
    CTA text</dds-text-cta
  >
</dds-content-section>
```

As part of the ongoing upgrade to Carbon for IBM.com v2, we have consolidated
some of the other `Content section` variants under a single one. Some of the
components that are compatible with the `Content section` component are:
`Content block simple`, `Card group`, `Carousel`, `Link list`.

### Content block simple

Note: With `Content block simple`, the `children-custom-class` attribute will need to be provided with proper grid class as shown below in order for the component to render correctly within the grid.

```html
<dds-content-section children-custom-class="bx--col-lg-16 bx--no-gutter">
  <dds-content-section-heading>Heading</dds-content-section-heading>
  <dds-content-section-copy>Copy</dds-content-section-copy>
  <dds-text-cta slot="footer" cta-type="local" href="https://www.example.com">
    CTA text</dds-text-cta
  >
  <dds-content-block-simple>
    <dds-content-block-heading>Content block simple heading</dds-content-block-heading>
    <dds-content-block-copy size="sm">Content block simple copy</dds-content-block-copy>
    <dds-video-player-container
      slot="media"
      video-id="1_9h94wo6b"></dds-video-player-container>
    <dds-text-cta slot="footer" cta-type="jump" href="https://www.ibm.com"
      >Jump to AI ethics and trust</dds-text-cta
    >
  </dds-content-block-simple>
</dds-content-section>
```

## Stable selectors

See
[our README](https://github.com/carbon-design-system/carbon-for-ibm-dotcom/tree/main/packages/web-components#stable-selectors-for-analytics-and-integratione2e-testing-in-web-components)

| Web Components selector         | Compatibility selector                        | Description |
| ------------------------------- | --------------------------------------------- | ----------- |
| `<dds-content-section>`         | `data-autoid="dds--content-section"`          | Component   |
| `<dds-content-section-copy>`    | `data-autoid="dds--content-section__copy"`    | Component   |
| `<dds-content-section-heading>` | `data-autoid="dds--content-section__heading"` | Component   |
