## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/button-group/index.js';
import '@carbon/web-components/es/icons/arrow--right/20.js';
```

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<dds-button-group>
  <dds-button-group-item href="https://example.com" cta-type="local">
    Lorem ipsum
  </dds-button-group-item>
  <dds-button-group-item href="https://example.com" cta-type="local">
    Dolor sit amet
  </dds-button-group-item>
</dds-button-group>
```

### click handler example for web components

```html
<dds-button-group>
  <dds-button-group-item @click="${this._handleClick}"></dds-button-group-item>
</dds-button-group>
```

### click handler example for HTML

```html
<dds-button-group>
  <dds-button-group-item onclick="handleClick()"></dds-button-group-item>
</dds-button-group>
```
