## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/button-group/index.js';
import '@carbon/web-components/es/icons/arrow--right/20.js';
```

<Markdown>{`${cdnJs({ components: ['button-group'] })}`}</Markdown>

<Markdown>{`${cdnCss()}`}</Markdown>

### HTML

```html
<c4d-button-group>
  <c4d-button-group-item href="https://example.com" cta-type="local">
    Lorem ipsum
  </c4d-button-group-item>
  <c4d-button-group-item href="https://example.com" cta-type="local">
    Dolor sit amet
  </c4d-button-group-item>
</c4d-button-group>
```

### click handler example for web components

```html
<c4d-button-group>
  <c4d-button-group-item @click="${this._handleClick}"></c4d-button-group-item>
</c4d-button-group>
```

### click handler example for HTML

```html
<c4d-button-group>
  <c4d-button-group-item onclick="handleClick()"></c4d-button-group-item>
</c4d-button-group>
```
