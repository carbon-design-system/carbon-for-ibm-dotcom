## Using a video player

The `<C4DVideoCTAContainer>` component lets you integrate
`<C4DTextCTA>`/`<C4DButtonCTA>`/`<C4DCardCTA>`/`<C4DFeatureCTA>` with video
player quickly. Only one `<C4DVideoCTAContainer>` has to be put in application.
Ensure that `<C4DVideoCTAContainer>` contains all
`<C4DTextCTA>`/`<C4DButtonCTA>`/`<C4DCardCTA>`/`<C4DFeatureCTA>`:

### JS

```javascript
import C4DTextCTA from '@carbon/ibmdotcom-web-components/es/components-react/cta/text-cta.js';
import C4DVideoCTAContainer from '@carbon/ibmdotcom-web-components/es/components-react/cta/video-cta-container';

function App() {
  return (
    <C4DVideoCTAContainer>
      <C4DTextCTA cta-type="video" href="0_ibuqxqbe"></C4DTextCTA>
    </C4DVideoCTAContainer>
  );
}
```

### `cds-cta-run-action` customization

For more flexible solution that is better tailored to your application's needs,
CTA components fire `cds-cta-run-action` custom event when user clicks on a CTA
with `type="video"`. Application code can take care of creating/putting our
video player component and working with our video player service API to put the
video data into the video player component:

```javascript
import KalturaPlayerAPI from '@carbon/ibmdotcom-services/es/services/KalturaPlayer/KalturaPlayer.js';
import '@carbon/ibmdotcom-web-components/es/components/cta/text-cta.js.js';
import '@carbon/ibmdotcom-web-components/es/components/expressive-modal/index.js';
import '@carbon/ibmdotcom-web-components/es/components/lightbox-media-viewer/lightbox-video-player.js';

...

document.addEventListener('cds-cta-run-action', async event => {
  const videoId = event.detail.href;
  const lightboxId = `lightbox-${videoId}`;
  let lightboxElem = document.getElementById(lightboxId);
  if (!lightboxElem) {
    lightboxElem = document.createElement('cds-expressive-modal');
    lightboxElem.size = 'full-width';
    lightboxElem.innerHTML = `
      <cds-expressive-modal-close-button></cds-expressive-modal-close-button>
      <cds-lightbox-video-player id="lightbox-video-player-${videoId}"></cds-lightbox-video-player>
    `;
    document.body.appendChild(lightboxElem);
    const videoTarget = document.createElement('div');
    videoTarget.id = `video-target-${videoId}`;
    videoTarget.className = 'cds--video-player__video';
    const videoPlayer = document.getElementById(`lightbox-video-player-${videoId}`);
    videoPlayer.appendChild(videoTarget);
    const [videoData] = await Promise.all([
      KalturaPlayerAPI.api(videoId),
      KalturaPlayerAPI.embedMedia(videoId, videoTarget.id, true),
    ]);
    const { name, description, duration } = videoData;
    videoPlayer.name = name;
    videoPlayer.duration = duration;
    videoPlayer.description = description;
  }
  lightboxElem.open = true;
});
```
