# `<c4d-content-block-simple>`

> The Content Block - Simple pattern is a variant of `<c4d-content-block>`, and
> includes a single content item, optional media (image), and ends with a CTA.

## Getting started

Here's a quick example to get you started.

## With text CTA

### JS (via import)

```javascript

// with text CTA
```
