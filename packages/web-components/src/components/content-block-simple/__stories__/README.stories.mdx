# `<dds-content-block-simple>`

> The Content Block - Simple pattern is a variant of `<dds-content-block>`, and
> includes a single content item, optional media (image), and ends with a CTA.

## Getting started

Here's a quick example to get you started.

| `<dds-content-block-copy>`    | `data-autoid="dds--content-block__copy"`    | Component   |
| `<dds-content-block-heading>` | `data-autoid="dds--content-block__heading"` | Component   |
