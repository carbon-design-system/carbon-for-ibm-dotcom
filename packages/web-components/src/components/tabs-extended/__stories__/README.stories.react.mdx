## Getting started

Here's a quick example to get you started.

### JS

```javascript
import C4DTabsExtended from '@carbon/ibmdotcom-web-components/es/components-react/tabs-extended/tabs-extended.js';
import C4DTab from '@carbon/ibmdotcom-web-components/es/components-react/tabs-extended/tab.js';

function App() {
  return (
    <C4DTabsExtended>
      <C4DTab label="First tab">
        <p>Content for first tab goes here.</p>
      </C4DTab>
      <C4DTab label="Second tab">
        <p>Content for second tab goes here.</p>
      </C4DTab>
      <C4DTab label="Third tab">
        <p>Content for third tab goes here.</p>
      </C4DTab>
    </C4DTabsExtended>
  );
}
```

## Setting active tab

To set a different default tab to be active upon load, set the `active-tab`
attribute to the index of your desired tab.

Here's an example targetting the third tab. Note that the indexing starts at 0,
which means the third tab is the one at index 2.

```javascript
function App() {
  return (
    <C4DTabsExtended active-tab="2">
      <C4DTab label="First tab">
        <p>Content for first tab goes here.</p>
      </C4DTab>
      <C4DTab label="Second tab">
        <p>Content for second tab goes here.</p>
      </C4DTab>
      <C4DTab label="Third tab">
        <p>Content for third tab goes here.</p>
      </C4DTab>
    </C4DTabsExtended>
  );
}
```
