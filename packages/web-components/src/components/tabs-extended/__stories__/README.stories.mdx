### HTML

```html
<cds-tabs-extended orientation="horizontal">
  <cds-tab label="First tab">
    <p>Content for first tab goes here.</p>
  </cds-tab>
  <cds-tab label="Second tab">
    <p>Content for second tab goes here.</p>
  </cds-tab>
  <cds-tab label="Third tab" selected="true">
    <p>Content for third tab goes here.</p>
  </cds-tab>
  <cds-tab label="Fourth tab">
    <p>Content for fourth tab goes here.</p>
  </cds-tab>
  <cds-tab label="Fifth tab" disabled="true">
    <p>Content for fifth tab goes here.</p>
  </cds-tab>
</cds-tabs-extended>
```

## Setting active tab

To set a different default tab to be active upon load, set the `active-tab`
attribute to the index of your desired tab.

Here's an example targetting the third tab. Note that the indexing starts at 0,
which means the third tab is the one at index 2.

```html
<cds-tabs-extended active-tab="2" orientation="horizontal">
  <cds-tab label="First tab">
    <p>Content for first tab goes here.</p>
  </cds-tab>
  <cds-tab label="Second tab">
    <p>Content for second tab goes here.</p>
  </cds-tab>
  <cds-tab label="Third tab" selected="true">
    <p>Content for third tab goes here.</p>
  </cds-tab>
  <cds-tab label="Fourth tab">
    <p>Content for fourth tab goes here.</p>
  </cds-tab>
  <cds-tab label="Fifth tab" disabled="true">
    <p>Content for fifth tab goes here.</p>
  </cds-tab>
</cds-tabs-extended>
```
