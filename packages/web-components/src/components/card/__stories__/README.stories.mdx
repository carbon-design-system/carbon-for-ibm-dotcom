## Card with pictogram

You can browse the [Carbon Pictogram library](https://www.carbondesignsystem.com/guidelines/pictograms/library/) to find a pictogram that fits your design.
You can also check out the repo [here](https://github.com/carbon-design-system/carbon/tree/main/packages/pictograms)

```html
<dds-card pictogram-placement="${pictogramPlacement}" href="https://example.com">
  <dds-card-eyebrow>Eyebrow text</dds-card-eyebrow>
  <dds-card-heading>Lorem ipsum dolor</dds-card-heading>
  <p>Copy to be shown on hover if the pictogram placement is bottom</p>

    slot="pictogram"
```

## Card - static

When there is no URL being passed to the `DDSCard`'s href attribute, the static variation will be rendered,
where the card is no longer clickable and optional `Tag` elements can also be added.

```html
<dds-card>
  <dds-card-heading>Lorem ipsum dolor sit amet</dds-card-heading>
  ${copy ? html`
  <p>${copy}</p>
  ` : ``}
  <dds-tag-group>
    <bx-tag>
      Most popular
    </bx-tag>
    <bx-tag type="purple">
      Enterprise
    </bx-tag>
  </dds-tag-group>
  <dds-card-footer href="https://example.com" icon-placement="right">
    Card CTA text
    <svg
      slot="icon"
      focusable="false"
      preserveAspectRatio="xMidYMid meet"
      xmlns="http://www.w3.org/2000/svg"
      fill="currentColor"
      aria-hidden="true"
      width="20"
      height="20"
      viewBox="0 0 20 20"
    >
      <path d="M11.8 2.8L10.8 3.8 16.2 9.3 1 9.3 1 10.7 16.2 10.7 10.8 16.2 11.8 17.2 19 10z"></path>
    </svg>
  </dds-card-footer>
</dds-card>
```

## Card - logo

If you need to use a Logo image for the Card component, use the Logo variant to ensure the Card
properly accomodates to the Logo image.

```html
<dds-card border logo href="https://example.com">
  <dds-image-logo
    slot="image"
    alt="Image logo alt text"
    default-src="https://fpoimg.com/320x160?text=2:1&bg_color=ee5396&text_color=161616"
  ></dds-image-logo>
  <dds-card-eyebrow>Microsoft</dds-card-eyebrow>
  <p>Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</p>
  <dds-tag-group>
    <bx-tag>
      Most popular
    </bx-tag>
    <bx-tag type="purple">
      Enterprise
    </bx-tag>
  </dds-tag-group>
  <dds-card-footer href="https://example.com"></dds-card-footer>
</dds-card>
```

## Light theme

An optional light theme is available under `color-scheme`.

```html
<dds-card color-scheme="light"></dds-card>
```

## Card with image

To add image to the `dds-card`, import the `image` component and place `dds-image` within `dds-card`:

```javascript
import '@carbon/ibmdotcom-web-components/es/components/image/index.js';
```

```html
<dds-card>
  <dds-card-heading>Lorem ipsum dolor sit amet</dds-card-heading>
  <dds-image slot="image" alt="alt text" default-src="../image.png"></dds-image>
  <dds-tag-group>
    <bx-tag>
      Most popular
```
