## Getting started

Here's a quick example to get you started.

### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/card/card';
import ArrowRight20 from 'carbon-web-components/es/icons/arrow--right/20';
```

or with [JSPM](https://jspm.org)

```html
<script type="module">
  import 'https://jspm.dev/@carbon/ibmdotcom-web-components@canary/es/components/card/card.js';
</script>
```

### HTML

```html
<dds-card href="https://example.com">
    <slot slot="eyebrow">eyebrow text</slot>
    <slot slot="heading">Lorem ipsum dolor sit amet</slot>
    <dds-card-footer slot="footer">
      Card cta text
      <svg slot="icon" />
    </dds-card-footer>
</dds-card>
```
