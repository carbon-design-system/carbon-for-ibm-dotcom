## Card with pictogram

You can browse the [Carbon Pictogram library](https://www.carbondesignsystem.com/guidelines/pictograms/library/) to find a pictogram that fits your design.
You can also check out the repo [here](https://github.com/carbon-design-system/carbon/tree/main/packages/pictograms)

```html
<dds-card
  pictogram-placement="${pictogramPlacement}"
  href="https://example.com"
>
  <dds-card-eyebrow>Eyebrow text</dds-card-eyebrow>
  <dds-card-heading>Lorem ipsum dolor</dds-card-heading>
  <p>Copy to be shown on hover if the pictogram placement is bottom</p>
  <svg
    slot="pictogram"
    focusable="false"
    preserveAspectRatio="xMidYMid meet"
    xmlns="http://www.w3.org/2000/svg"
    stroke="currentColor"
    data-autoid="dds--card__pictogram"
    aria-label="Pictogram description"
    width="48"
    height="48"
    viewBox="0 0 48 48"
    role="img"
    class="bx--card__pictogram"
  >
    <path
      fill="none"
      stroke-linejoin="round"
      stroke-miterlimit="10"
      stroke-width=".72"
      d="M37,32 H11c-1.1,0-2-0.9-2-2V13c0-1.1,0.9-2,2-2h26c1.1,0,2,0.9,2,2v17C39,31.1,38.1,32,37,32z M17,37h14 M24,32v5 M9,27h30"
    >
    </path>
  </svg>
</dds-card>
```

## Card - static

When there is no URL being passed to the `DDSCard`'s href attribute, the static variation will be rendered,
where the card is no longer clickable and optional `Tag` elements can also be added.

```html
<dds-card>
  <dds-card-heading>Lorem ipsum dolor sit amet</dds-card-heading>
  <dds-tag-group>
    <bx-tag>
      Most popular
    </bx-tag>
    <bx-tag type='purple'>
      Enterprise
    </bx-tag>
  </dds-tag-group>
  ${copy
    ? html`
        <p>${copy}</p>
      `
    : ``}
  <dds-card-footer href='https://example.com' icon-placement="right">
    Card CTA text
    <svg
      slot="icon"
      focusable="false"
      preserveAspectRatio="xMidYMid meet"
      xmlns="http://www.w3.org/2000/svg"
      fill="currentColor"
      aria-hidden="true"
      width="20"
      height="20"
      viewBox="0 0 20 20"
    >
      <path d="M11.8 2.8L10.8 3.8 16.2 9.3 1 9.3 1 10.7 16.2 10.7 10.8 16.2 11.8 17.2 19 10z"></path>
    </svg>
  </dds-card-footer>
</dds-card>
```
