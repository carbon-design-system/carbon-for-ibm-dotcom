## Getting started

Here's a quick example to get you started.

### JS

```javascript

  return (
    <C4DLeadspaceWithSearch>
      <C4DLeadspaceHeading>
        Find a product
      </C4DLeadspaceHeading>
        <C4DLeadspaceWithSearchCopy>
          Automate your software release process with continuous delivery
```
