## Getting started

### Quick start

`<c4d-dotcom-shell-container>` automatically renders contents of dotcom shell
from default service data.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/dotcom-shell/index.js';
```

<Description markdown={`${cdnJs({ components: ['dotcom-shell'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<c4d-dotcom-shell-container>
  <main>(The main content)</main>
</c4d-dotcom-shell-container>
```
