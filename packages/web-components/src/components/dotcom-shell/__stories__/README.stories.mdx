## Getting started

### Quick start

`<dds-dotcom-shell-container>` automatically renders contents of dotcom shell from default service data.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/dotcom-shell/index.js';
```

<Description markdown={`${cdnJs({ components: ['dotcom-shell'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<dds-dotcom-shell-container>
  <main>(The main content)</main>
</dds-dotcom-shell-container>
```
