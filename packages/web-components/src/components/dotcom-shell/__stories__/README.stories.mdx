## Getting started

### Quick start

`<dds-dotcom-shell-container>` automatically renders contents of dotcom shell from default service data.

##### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/dotcom-shell/dotcom-shell-container';
```

##### HTML

```html
<dds-dotcom-shell-container>
  <main>(The main content)</main>
</dds-dotcom-shell-container>
```

### Rendering dotcom shell content by your own

You can also render the footer content by your own, e.g. for server-side rendering, like:

##### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/dotcom-shell/dotcom-shell';
```

##### HTML

```html
(The masthead content)
<dds-dotcom-shell>
  <main>(The main content)</main>
</dds-dotcom-shell>
(The footer content)
```

> ðŸ’¡ See [masthead doc](/?path=/docs/components-masthead--default) for generating the content for masthead, and [footer doc](/?path=/docs/components-footer--default) for generating the content for footer.
