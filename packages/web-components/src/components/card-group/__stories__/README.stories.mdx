### HTML (with Mixed Media)

In order to have video components, make sure to import `video-cta-container` and place `dds-card-group` within.

To set the `dds-card-in-card` to play video, see more details [here](https://www.ibm.com/standards/carbon/web-components/?path=/docs/components-card-in-card--default).

```javascript
import '@carbon/ibmdotcom-web-components/es/components/cta/video-cta-container.js';
```

```html
<dds-video-cta-container>
  <dds-card-group>
    <dds-card-group-item cta-type="video" href="1_9h94wo6b">
      <dds-card-eyebrow>Topic</dds-card-eyebrow>
      <dds-card-cta-footer cta-type="video" slot="footer" href="1_9h94wo6b"> </dds-card-cta-footer>
    </dds-card-group-item>
  </dds-card-group>
</dds-video-cta-container>
```
