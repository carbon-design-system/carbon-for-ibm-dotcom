### HTML (with Mixed Media)

In order to have video components, make sure to import `video-cta-container` and
place `cds-card-group` within.

To set the `cds-card-in-card` to play video, see more details
[here](https://www.ibm.com/standards/carbon/web-components/?path=/docs/components-card-in-card--default).

```javascript
import '@carbon/ibmdotcom-web-components/es/components/cta/video-cta-container.js';
```

```html
<c4d-video-cta-container>
  <c4d-card-group>
    <c4d-card-group-item cta-type="video" href="0_ibuqxqbe">
      <c4d-card-eyebrow>Topic</c4d-card-eyebrow>
      <c4d-card-footer cta-type="video" slot="footer" href="0_ibuqxqbe">
      </c4d-card-footer>
    </c4d-card-group-item>
  </c4d-card-group>
</c4d-video-cta-container>
```

### Grid mode

There are three different options for the `grid-mode` property:

- Default (32px gap)
- Condensed (16px gap)
- Collapsed (no gap)
