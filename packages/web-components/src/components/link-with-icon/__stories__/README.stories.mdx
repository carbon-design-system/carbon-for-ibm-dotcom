# `<dds-link-with-icon>`

> The `<dds-link-with-icon>` component should be used primarily as a navigational element,
> with an icon as an indicator to the type of content being referenced, e.g.
> url, external url, file.

## Getting started

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/link-with-icon/link-with-icon.js';
```

### JS (via CDN)

```html
<!-- LATEST tag -->
<script type="module" src="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/latest/link-with-icon.min.js"></script>

<!-- NEXT tag -->
<script type="module" src="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/next/link-with-icon.min.js"></script>

<!-- BETA tag -->
```
