## Getting started

Here's a quick example to get you started.

### JS

```javascript
import C4DSearchWithTypeahead from '@carbon/ibmdotcom-web-components/es/components-react/search-with-typeahead/search-with-typeahead.js';

function App() {
  return <C4DSearchWithTypeahead></C4DSearchWithTypeahead>;
}
```

To have the Search open on load, and always be active, you can render the app as
follows:

```javascript
function App() {
  return (
    <C4DSearchWithTypeahead
      should-remain-open={true}
      active={true}></C4DSearchWithTypeahead>
  );
}
```
