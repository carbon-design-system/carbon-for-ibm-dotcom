## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/card-section-simple/index.js';
```

### HTML

```html
<cds-card-section-simple>
  <cds-content-section-heading
    >Aliquam condimentum interdum</cds-content-section-heading
  >
  <cds-card-group>
    <cds-card-group-item href="https://example.com" cta-type="local">
      <cds-card-heading>Nunc convallis lobortis</cds-card-heading>
      <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean et
        ultricies est. Mauris iaculis eget dolor nec hendrerit. Phasellus at
        elit sollicitudin, sodales nulla quis, consequat libero.
      </p>
      <cds-card-cta-footer slot="footer"> </cds-card-cta-footer>
    </cds-card-group-item>
  </cds-card-group>
</cds-card-section-simple>
```

### HTML (with images)

```html

  <cds-card-group>
    <cds-card-group-item href="https://example.com" cta-type="local">
```
