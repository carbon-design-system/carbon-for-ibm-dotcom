## Custom Event

This event will return the field name and value. The values need to be sent to
MRS. For more info, check >
[W3 Publisher](https://w3.ibm.com/w3publisher/urx/notice-and-choice)

```javascript
document.addEventListener('cds-notice-choice-change', (event) => {
  this.handleInputChange(event.detail.field, event.detail.value);
});
```
