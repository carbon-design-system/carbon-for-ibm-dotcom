# `<dds-promo-item>`

> The Promo Item - is a simple promo text component that provides different variants for showcasing images, statistics, and pictograms content.

## Getting started

Here's a quick example to get you started.

### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/card/card-heading.js';
import '@carbon/ibmdotcom-web-components/es/components/content-item/content-item-heading.js';
import '@carbon/ibmdotcom-web-components/es/components/promo-item/promo-item.js';

// With Image.
import '@carbon/ibmdotcom-web-components/es/components/image/image.js';

// With Statistic.
import '@carbon/ibmdotcom-web-components/es/components/promo-item/promo-item-statistic.js';
```
