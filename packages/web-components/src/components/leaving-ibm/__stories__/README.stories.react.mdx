## Getting started

Here's a quick example to get you started.

### JS

> ðŸ’¡ `Provider` is only in the context of the example, and can be swapped out with other implementations based on the application setup.

```javascript
import DDSLeavingIBMContainer from '@carbon/ibmdotcom-web-components/es/components-react/leaving-ibm/leaving-ibm-container';
import DDSLinkWithIcon from '@carbon/ibmdotcom-web-components/es/components-react/link-with-icon/link-with-icon';
import Launch20 from '@carbon/icons-react/es/launch/20';

function App() {
  return (
    // Add component to
    <DDSLeavingIBMContainer />

    ...

    // Trigger modal on click
    <DDSLinkWithIcon iconPlacement="right" href={href} data-leaving-ibm>
      Learn more about the Carbon Design System
      <Launch20 slot="icon" />
    </DDSLinkWithIcon>
  );
}
```

> ðŸ’¡ Give your link the `data-leaving-ibm` attribute to trigger the modal on click. Alternatively, you can use the [custom events available to the modal](https://web-components.carbondesignsystem.com/?path=/docs/components-modal--default#bx-modal-attributes-and-properties) to create custom triggering behavior.
