## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/leaving-ibm/index';
```

<Description markdown={`${cdnJs({ components: ['leaving-ibm'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<!-- Add component to document -->
<dds-leaving-ibm-container></dds-leaving-ibm-container>

...

<!-- Trigger modal on click -->
<dds-link-with-icon href="https://www.carbondesignsystem.com/all-about-carbon/what-is-carbon/" data-leaving-ibm>
  Learn more about the Carbon Design System
</dds-link-with-icon>
```

> ðŸ’¡ Give your link the `data-leaving-ibm` attribute to trigger the modal on click. Alternatively, you can use the [custom events available to the modal](https://web-components.carbondesignsystem.com/?path=/docs/components-modal--default#bx-modal-attributes-and-properties) to create custom triggering behavior.
