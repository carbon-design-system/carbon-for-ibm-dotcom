# `<cds-content-block-card-static>`

> The Content Block - Card Static pattern is a variant of `<cds-content-block>`,
> and includes a card group, a single content item, and ends with a CTA.

[![Edit @carbon/ibmdotcom-web-components](https://codesandbox.io/static/img/play-codesandbox.svg)](https://githubbox.com/carbon-design-system/carbon-for-ibm-dotcom/tree/main/packages/web-components/examples/codesandbox/components/content-block-card-static)

## Feature Flag

This component is currently behind a feature flag, in order to use it you'll
need to add the following into your `.env` file.

```
  C4D_CONTENT_BLOCK_CARD_STATIC=true
```

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/content-block-card-static/index.js';

// with tags
import '@carbon/web-components/es/components/tag/tag.js';

// Optional icons for Button Group
import ArrowRight20 from '@carbon/web-components/es/icons/arrow--right/20.js';
import Chat20 from '@carbon/web-components/es/icons/chat/20.js';
```
