> The Content Block - Card Static pattern is a variant of `<dds-content-block>`, and
> includes a card group, a single content item, and ends with a CTA.

## Getting started

Here's a quick example to get you started.

##### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/content-block-media/index.js';

// with tags
import 'carbon-web-components/es/components/tag/tag';

// Optional icons for Button Group
import ArrowRight20 from 'carbon-web-components/es/icons/arrow--right/20.js';
import Chat20 from 'carbon-web-components/es/icons/chat/20.js';
```

### HTML

```html

        <dds-tag-group>
```
