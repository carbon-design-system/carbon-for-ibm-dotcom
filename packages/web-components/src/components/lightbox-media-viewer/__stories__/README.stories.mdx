# Lightbox Media Viewer

> This launches a modal containing an image or video, and a description
> detailing the media content.

## Getting started

### JS (via import)

```javascript
import '@carbon/ibmdotcom-web-components/es/components/lightbox-media-viewer/lightbox-media-viewer.js';

// Opens the modal containing the lightbox image viewer
document.getElementById('my-lightbox').open = true;
```

## Using Images

### HTML

```html
<dds-expressive-modal id="my-lightbox" expressive-size="full-width">
  <dds-expressive-modal-close-button></dds-expressive-modal-close-button>
  <dds-lightbox-media-viewer
    alt="Image alt text"
    default-src="https://fpoimg.com/1280x720?text=16:9&bg_color=ee5396&text_color=161616"
    description="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean et ultricies est.Mauris iaculis eget dolor nec hendrerit."
    title="Curabitur malesuada varius mi eu posuere"
  >
  </dds-lightbox-media-viewer>
</dds-expressive-modal>
```

## Using Videos

### HTML

```html
<dds-expressive-modal id="my-lightbox" expressive-size="full-width">
  <dds-expressive-modal-close-button></dds-expressive-modal-close-button>
  <dds-lightbox-media-viewer id="my-video" video-id="1_9h94wo6b"></dds-lightbox-media-viewer>
</dds-expressive-modal>
```

## Stable selectors

See [our README](https://github.com/carbon-design-system/carbon-for-ibm-dotcom/tree/main/packages/web-components#stable-selectors-for-analytics-and-integratione2e-testing-in-web-components) to see how Web Components selector and `data-autoid` should be used.

| Web Components selector                                | Compatibility selector                                     | Description |
| ------------------------------------------------------ | ---------------------------------------------------------- | ----------- |
| `<dds-expressive-modal>`                               | `data-autoid="dds--expressive-modal"`                      | Component   |
| `::part(title)` in `<dds-lightbox-media-viewer>`       | `data-autoid="dds--Lightbox-media-viewer__content__title"` | Interactive |
| `::part(description)` in `<dds-lightbox-media-viewer>` | `data-autoid="dds--Lightbox-media-viewer__content__desc"`  | Interactive |
