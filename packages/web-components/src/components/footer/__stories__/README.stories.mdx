## Getting started

### Quick start

`<dds-footer-container>` automatically renders contents of footer from default service data. If you want change in services/data integration, etc., see ["Customizing data integration" section](#customizing-data-integration) below.

##### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/footer/footer-container';
```

##### HTML

```html
<dds-footer-container></dds-footer-container>
```

> ðŸ’¡ You can get short variant of footer with `size="short"` attribute.

### Customizing service/data integration

The Footer takes advantage of a redux store which manages the API calls made
to fetch translation data, locale data, etc. To manually initialize the Redux
store (e.g. integrating with your application level redux store), the underlying
component, `<dds-footer-composite>`, can be initialized instead:


##### JS

```javascript
import { createStore, applyMiddleware, combineReducers } from 'redux';
import thunkMiddleware from 'redux-thunk';
import { createLogger } from 'redux-logger';
// The default reducers, you can e.g. add your own reducer to `reducers` table
import * as reducers from '@carbon/ibmdotcom-web-components/es/globals/services-store/reducers';
import ConnectMixin from '@carbon/ibmdotcom-web-components/es/globals/mixins/connect';
import {
  mapStateToProps,
  mapDispatchToProps,
} from '@carbon/ibmdotcom-web-components/es/components/footer/footer-container';
import DDSFooterComposite from '@carbon/ibmdotcom-web-components/es/components/footer/footer-composite';

// Creates `<app-footer-container>` custom element that wires your store to our rendering logic
customElements.define(
  'app-footer-container',
  class extends ConnectMixin(
    store,
    mapStateToProps,
    mapDispatchToProps
  )(DDSFooterComposite) {}
);
```

// Connects `<dds-footer-composite>` to `@carbon/ibmdotcom-servies` and creates `<myapp-footer-container>`
customElements.define('myapp-footer-container', class extends ConnectMixin(store, mapStateToProps, mapDispatchToProps)(DDSFooterComposite) {})
```

##### HTML

```html
<myapp-footer-container></myapp-footer-container>
```

### Rendering footer content by your own

You can also render the footer content by your own, e.g. for server-side rendering, like:

##### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/footer/footer';
import '@carbon/ibmdotcom-web-components/es/components/footer/footer-logo';
import '@carbon/ibmdotcom-web-components/es/components/footer/footer-nav';
import '@carbon/ibmdotcom-web-components/es/components/footer/footer-nav-group';
import '@carbon/ibmdotcom-web-components/es/components/footer/footer-nav-item';
import '@carbon/ibmdotcom-web-components/es/components/footer/legal-nav';
import '@carbon/ibmdotcom-web-components/es/components/footer/legal-nav-item';
import '@carbon/ibmdotcom-web-components/es/components/footer/locale-button';

document.addEventListener('click', event => {
  if (event.target.matches('dds-locale-button')) {
    document.querySelector('dds-locale-modal').open = true;
  }
});
```

##### HTML

```html
<dds-footer>
  <dds-footer-logo></dds-footer-logo>
  <dds-footer-nav>
    <dds-footer-nav-group title-text="Discover">
      <dds-footer-nav-item href="https://www.ibm.com/products">Products</dds-footer-nav-item>
      <dds-footer-nav-item href="https://www.redbooks.ibm.com/">Redbooks</dds-footer-nav-item>
      <dds-footer-nav-item href="https://www.ibm.com/services">Services</dds-footer-nav-item>
    </dds-footer-nav-group>
    <dds-footer-nav-group title-text="Information for...">
      <dds-footer-nav-item href="https://developer.ibm.com/">Developers</dds-footer-nav-item>
      <dds-footer-nav-item href="https://www.ibm.com/partnerworld/public">Business</dds-footer-nav-item>
    </dds-footer-nav-group>
  </dds-footer-nav>
  <dds-locale-button></dds-locale-button>
  <dds-legal-nav>
    <dds-legal-nav-item href="https://www.ibm.com/contact/us/en/>Contact IBM</dds-legal-nav-item>
    <dds-legal-nav-item href="https://www.ibm.com/privacy/us/en/">Privacy</dds-legal-nav-item>
    <dds-legal-nav-item href="https://www.ibm.com/us-en/legal">Terms of use</dds-legal-nav-item>
  </dds-legal-nav>
</dds-footer>
```

> ðŸ’¡ See [`<dds-locale-modal>` doc](/?path=/docs/components-locale-modal--default) for generating the content for locale modal.
