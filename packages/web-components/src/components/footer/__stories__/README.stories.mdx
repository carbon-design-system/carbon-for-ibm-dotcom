### Language selector

The option to use a language selector is available in lieu of the locale
button/selector.

The following attributes are needed:

| Attribute                 | Description                             |
| ------------------------- | --------------------------------------- |
| `language-selector-label` | `The input box placeholder`             |
| `selected-language`       | `The initial selected language`         |
| `langList`                | `Object containing available languages` |

Example implementation:

```html
<c4d-footer-container
  size="regular"
  language-selector-label="Choose a language"
  clear-selection-label="Clear language selection"
  selected-language="English"></c4d-footer-container>
```

```javascript
const languageList = [
  { id: 'da', text: 'Danish / Dansk' },
  { id: 'nl', text: 'Dutch / Nederlands' },
  { id: 'en', text: 'English' },
];
document.querySelector('cds-footer-container').langList = languageList;
```
