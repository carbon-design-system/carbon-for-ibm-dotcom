## Getting started

##### JS

```javascript
import ConnectMixin from '@carbon/ibmdotcom-web-components/es/globals/mixins/connect';
import DDSFooterComposite from '@carbon/ibmdotcom-web-components/es/components/footer/footer-composite';
import { store, mapStateToProps, mapDispatchToProps } from '@carbon/ibmdotcom-web-components/es/components/footer/footer-container';

// Wires `<dds-footer-composite>` to `@carbon/ibmdotcom-servies` and creates `<app-footer-container>`
customElements.define('app-footer-container', class extends ConnectMixin(store, mapStateToProps, mapDispatchToProps)(DDSFooterComposite) {})
```

##### HTML

```html
<app-footer-container></app-footer-container>
```

## `<dds-footer-composite>`

`<dds-footer-composite>` automatically renders contents of footer from the given data.
You can populate the data automatically from `@carbon/ibmdotcom-servies` by mixing a Redux store into this custom element class, shown in the example in Getting started section above.

## `<dds-footer-nav>`

`<dds-footer-nav>` works as the primary collection of links in footer.
The markup should be like:

```html
<dds-footer-nav>
  <dds-footer-nav-group title-text="Discover">
    <dds-footer-nav-item href="https://www.ibm.com/products">Products</dds-footer-nav-item>
    <dds-footer-nav-item href="https://www.redbooks.ibm.com/">Redbooks</dds-footer-nav-item>
    <dds-footer-nav-item href="https://www.ibm.com/services">Services</dds-footer-nav-item>
  </dds-footer-nav-group>
  <dds-footer-nav-group title-text="Information for...">
    <dds-footer-nav-item href="https://developer.ibm.com/">Developers</dds-footer-nav-item>
    <dds-footer-nav-item href="https://www.ibm.com/partnerworld/public">Business</dds-footer-nav-item>
  </dds-footer-nav-group>
</dds-footer-nav>
```

## `<dds-legal-nav>`

`<dds-legal-nav>` works as the secondary collection of links in footer.

The markup should be like:

```html
<dds-legal-nav>
  <dds-legal-nav-item href="https://www.ibm.com/contact/us/en/>Contact IBM</dds-legal-nav-item>
  <dds-legal-nav-item href="https://www.ibm.com/privacy/us/en/">Privacy</dds-legal-nav-item>
  <dds-legal-nav-item href="https://www.ibm.com/us-en/legal">Terms of use</dds-legal-nav-item>
</dds-legal-nav>
```

## Stable selectors

See [our README](https://github.com/carbon-design-system/ibm-dotcom-library/tree/master/packages/web-components#stable-selectors-for-analytics-and-integratione2e-testing-in-web-components) to see how Web Components selector and `data-autoid` should be used.

| Web Components selector  | Compatibility selector                 | Description |
| ------------------------ | -------------------------------------- | ----------- |
| `<dds-footer>`           | `data-autoid="dds--footer"`            | Component   |
| `<dds-footer-nav>`       | `data-autoid="dds--footer-nav"`        | Component   |
| `<dds-footer-nav-group>` | `data-autoid="dds--footer-nav-group"`  | Component   |
| `<dds-footer-logo>`      | `data-autoid="dds--footer-logo"`       | Component   |
| `<dds-locale-button>`    | `data-autoid="dds--footer-locale-btn"` | Interactive |
| `<dds-legal-nav>`        | `data-autoid="dds--legal-nav"`         | Component   |
