## Getting started

Here's a quick example to get you started.

### JS

```javascript

function App() {
  return (
    <C4DFooterContainer
      lang-display='United States - English'
      language-selector-label='Choose a language'
      clear-selection-label='Clear language selection'
      selected-language='English'

  );
}
```
