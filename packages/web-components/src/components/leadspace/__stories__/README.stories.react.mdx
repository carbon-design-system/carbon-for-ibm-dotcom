## Getting started

Here's a quick example to get you started.

### JS

```javascript
import ArrowRight20 from '@carbon/icons-react/es/arrow--right/20.js';
import C4DBackgroundMedia from '@carbon/ibmdotcom-web-components/es/components-react/background-media/background-media';
import C4DButtonCTA from '@carbon/ibmdotcom-web-components/es/components-react/cta/button-cta';
import C4DButtonGroup from '@carbon/ibmdotcom-web-components/es/components-react/button-group/button-group';
import C4DButtonGroupItem from '@carbon/ibmdotcom-web-components/es/components-react/button-group/button-group-item';
import C4DLeadspace from '@carbon/ibmdotcom-web-components/es/components-react/leadspace/leadspace';
import C4DLeadspaceHeading from '@carbon/ibmdotcom-web-components/es/components-react/leadspace/leadspace-heading';
import C4DImageItem from '@carbon/ibmdotcom-web-components/es/components-react/image/image-item';
import C4DVideoPlayerContainer from '@carbon/ibmdotcom-web-components/es/components-react/video-player/video-player-container';

function App() {
  return (
    <C4DLeadspace size="tall">
      <C4DLeadspaceHeading>Heading can go on two lines max</C4DLeadspaceHeading>
      Use this area for a short line of copy to support the title
      <C4DButtonGroup slot="action">
        <C4DButtonGroupItem href="https://www.example.com">
          <ArrowRight20 slot="icon" />
          Button 1
        </C4DButtonGroupItem>
        <C4DButtonGroupItem href="https://www.example.com">
          <ArrowRight20 slot="icon" />
          Button 2
        </C4DButtonGroupItem>
      </C4DButtonGroup>
      <C4DBackgroundMedia
        slot="image"
        default-src="https://picsum.photos/id/1076/1056/480"
        alt="Image alt text"
        opacity="100">
        <C4DImageItem
          media="(min-width: 672px)"
          srcset="https://picsum.photos/id/1076/672/400"></C4DImageItem>
        <C4DImageItem
          media="(min-width: 0)"
          srcset="https://picsum.photos/id/1076/320/370"></C4DImageItem>
      </C4DBackgroundMedia>
    </C4DLeadspace>
  );
}
```

## Using a background video

It is possible to display a video instead of an image. To do so, use the
`video-player-container` component within the `background-media` componet as
such:

```javascript
function App() {
  return (
    <C4DLeadspace size="tall">
      <C4DLeadspaceHeading>Heading can go on two lines max</C4DLeadspaceHeading>
      Use this area for a short line of copy to support the title
      <C4DButtonGroup slot="action">
        <C4DButtonGroupItem href="https://www.example.com">
          <ArrowRight20 slot="icon" />
          Button 1
        </C4DButtonGroupItem>
        <C4DButtonGroupItem href="https://www.example.com">
          <ArrowRight20 slot="icon" />
          Button 2
        </C4DButtonGroupItem>
      </C4DButtonGroup>
      <C4DBackgroundMedia slot="image" alt="Image alt text" opacity="100">
        <C4DVideoPlayerContainer
          video-id="0_ibuqxqbe"
          background-mode="true"></C4DVideoPlayerContainer>
      </C4DBackgroundMedia>
    </C4DLeadspace>
  );
}
```

## Using video player CTA button

If using the video player cta in the Leadspace, make sure to follow the
guidelines in the
[CTA documentation](/?path=/docs/components-cta--text#using-a-video-player). As
the Leadspace reserves the "action" slot for its CTAs, make sure to include
`slot='action'` as an attribute to `video-cta-container`:

```javascript
function App() {
  return (
    <C4DLeadspace size="tall">
      <C4DLeadspaceHeading>Heading can go on two lines max</C4DLeadspaceHeading>
      Use this area for a short line of copy to support the title
      <C4DVideoCTAContainer slot="action">
        <C4DButtonGroup slot="action">
          <C4DButtonGroupItem cta-type="video" href="1_9h94wo6b">
            <ArrowRight20 slot="icon" />
            Button 1
          </C4DButtonCTA>
          <C4DButtonGroupItem href="https://www.example.com">
            <ArrowRight20 slot="icon" />
            Button 2
          </C4DButtonGroupItem>
        </C4DButtonGroup>
      </C4DVideoCTAContainer>
      <C4DBackgroundMedia
        slot="image"
        default-src="https://picsum.photos/id/1076/1056/480"
        alt="Image alt text"
        opacity="100">
        <C4DImageItem
          media="(min-width: 672px)"
          srcset="https://picsum.photos/id/1076/672/400"></C4DImageItem>
        <C4DImageItem
          media="(min-width: 0)"
          srcset="https://picsum.photos/id/1076/320/370"></C4DImageItem>
      </C4DBackgroundMedia>
    </C4DLeadspace>
  );
}
```

## Leadspace heading

The heading component has two attributes that can be used.

### Highlight
You can use the `highlight` attribute to highlight a specific word or phrase in your heading.

```javascript
function App() {
  return (
    <C4DLeadspace>
      <C4DLeadspaceHeading highlight="important">Leadspace with an important word</C4DLeadspaceHeading>
      Leadspace copy
    </C4DLeadspace>
  );
}
```

In the previous example the word "important" will be highlighted.

### Type style

With the `type-style` attribute, you can modify the type of the heading.

There are two options:
* display-01 (default)
* fluid-heading-05

```javascript
function App() {
  return (
    <C4DLeadspace>
      <C4DLeadspaceHeading type-style="fluid-heading-05">Heading</C4DLeadspaceHeading>
      Leadspace copy
    </C4DLeadspace>
  );
}
```

Note: The `short` variation of the leadspace can only have `fluid-heading-05` as its type heading.
