## Using a background video

It is possible to display a video instead of an image. To do so, use the
`video-player-container` component within the `background-media` componet as
such:

```html
<c4d-leadspace>
  <c4d-leadspace-heading>LeadSpace title</c4d-leadspace-heading>
  Leadspace copy
  <c4d-button-group slot="action">
    <c4d-button-group-item href="www.example.com">
      Button 1
      <svg ...>...</svg>
    </c4d-button-group-item>
    <c4d-button-group-item href="www.example.com">
      Button 2
      <svg ...>...</svg>
    </c4d-button-group-item>
  </c4d-button-group>
  <c4d-background-media slot="image" opacity="100">
    <c4d-video-player-container
      video-id="1_9h94wo6b"
      background-mode="true"></c4d-video-player-container>
  </c4d-background-media>
</c4d-leadspace>
```

## Using video player CTA button

If using the video player cta in the Leadspace, make sure to follow the
guidelines in the
[CTA documentation](/?path=/docs/components-cta--text#using-a-video-player). As
the Leadspace reserves the "action" slot for its CTAs, make sure to include
`slot='action'` as an attribute to `video-cta-container`:

```html
<c4d-leadspace alt="alt text" default-src="www.example.com">
  <c4d-leadspace-heading>LeadSpace title</c4d-leadspace-heading>
  LeadSpace copy
  <c4d-video-cta-container slot="action">
    <c4d-button-group slot="action">
      <c4d-button-group-item cta-type="video" href="1_9h94wo6b">
        Button 1
        <svg ...>...</svg>
      </c4d-button-group-item>
      <c4d-button-group-item href="www.example.com">
        Button 2
        <svg ...>...</svg>
      </c4d-button-group-item>
    </c4d-button-group>
  </c4d-video-cta-container>
  <c4d-background-media
    slot="image"
    opacity="100"
    alt="alt text"
```

## Leadspace heading

The heading component has two attributes that can be used.

### Highlight
You can use the `highlight` attribute to highlight a specific word or phrase in your heading.

```html
<c4d-leadspace alt="alt text" default-src="www.example.com">
  <c4d-leadspace-heading highlight="important">LeadSpace title with an important word</c4d-leadspace-heading>
  LeadSpace copy
</c4d-leadspace>
```

In the previous example the word "important" will be highlighted.

### Type style

With the `type-style` attribute, you can modify the type of the heading.

There are two options:
* display-01 (default)
* fluid-heading-05

```html
<c4d-leadspace alt="alt text" default-src="www.example.com">
  <c4d-leadspace-heading type-style="fluid-heading-05">LeadSpace title</c4d-leadspace-heading>
  LeadSpace copy
</c4d-leadspace>
```

Note: The `short` variation of the leadspace can only have `fluid-heading-05` as its type heading.
