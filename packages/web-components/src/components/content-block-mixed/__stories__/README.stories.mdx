
# `<dds-content-block-mixed>`

> The "Content Block - Mixed Groups" pattern leverages the Content Block as the overall container of this pattern, and allows only a few opinionated block patterns to be included.

## Getting started

Here's a quick example to get you started.

### HTML

```html
<dds-content-block-mixed>
  <dds-content-block-heading>Lorem ipsum dolor sit amet</dds-content-block-heading>
  <dds-content-block-paragraph>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean et ultricies est. Mauris iaculis eget dolor nec hendrerit.
    Phasellus at elit sollicitudin, sodales nulla quis, consequat libero.
  </dds-content-block-paragraph>

  <dds-content-block-cards>
    <dds-content-block-heading>Lorem ipsum dolor sit amet</dds-content-block-heading>
    <dds-card-group slot="content">
      <dds-card-group-item href="https://example.com">

        </p>

      </dds-image-item>
    </dds-image-with-caption>
    <dds-content-item copy="Lorem ipsum *dolor* sit amet">
      <dds-content-item-heading>Lorem ipsum dolor sit amet.</dds-content-item-heading>
    </dds-content-item>
    <dds-content-item copy="Lorem ipsum *dolor* sit amet">
      <dds-content-item-heading>Lorem ipsum dolor sit amet.</dds-content-item-heading>
    </dds-content-item>
    <dds-card-cta slot="footer" href="https://www.ibm.com/" type="local">
      Lorem ipsum dolor sit amet
      <dds-card-cta-footer>
        ${ArrowRight20({ slot: 'icon' })}

    </dds-card-cta>
  </dds-content-group-simple>
</dds-content-block-mixed>
```
