## Getting started

Here's a quick example to get you started.

### JS

```javascript
import '@carbon/ibmdotcom-web-components/es/components/back-to-top/index.js';

function App() {
  return (
    <main id="example-main-content">
      <DDSBackToTop />
    </main>
  );
}
```
