## `Misc attributes`

####   `should render with minimum attributes for narrow screen`

```
<button
  aria-controls="content"
  aria-expanded="false"
  class="cds--accordion__heading"
  type="button"
>
  <div class="cds--accordion__title">
    <slot name="title">
    </slot>
  </div>
</button>
<div
  class="cds--accordion__content"
  id="content"
>
  <ul>
    <slot>
    </slot>
  </ul>
</div>

```
