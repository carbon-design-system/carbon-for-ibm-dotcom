Code snippets are small blocks of reusable code that can be inserted in a code
file.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/code-snippet/index.js';
```

<Description markdown={`${cdnJs({ components: ['code-snippet'] })}`} />
<Description markdown={`${cdnCss()}`} />

### HTML

```html
<cds-code-snippet>
  node -v Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiis,
  veritatis voluptate id incidunt molestiae officia possimus, quasi itaque
  alias, architecto hic, dicta fugit? Debitis delectus quidem explicabo vitae
  laboriosam!
</cds-code-snippet>
```

## Single line

The Terminal style is for single-line.

```html
<cds-code-snippet>
  node -v Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiis,
  veritatis voluptate id incidunt molestiae officia possimus, quasi itaque
  alias, architecto hic, dicta fugit? Debitis delectus quidem explicabo vitae
  laboriosam!
</cds-code-snippet>
```

## Multi line

The Code style is for larger, multi-line code snippets.

```html
<cds-code-snippet type="multi">
  node -v Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiis,
  veritatis voluptate id incidunt molestiae officia possimus, quasi itaque
  alias, architecto hic, dicta fugit? Debitis delectus quidem explicabo vitae
  laboriosam!
</cds-code-snippet>
```

## Inline

The Inline style is for code used within a block of text.

```html
<cds-code-snippet type="inline"> node -v </cds-code-snippet>
```

## Skeleton

For the skeleton variation, utilize `<cds-code-snippet-skeleton>`.

```html
<cds-code-snippet-skeleton type="single"></cds-code-snippet-skeleton>
```
