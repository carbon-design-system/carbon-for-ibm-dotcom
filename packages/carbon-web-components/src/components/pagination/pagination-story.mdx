The pagination component is used to switch through multiple pages of items, when
only a maximum number of items can be displayed per page. Can be used in
combination with other components like data table.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/web-components/es/components/pagination/index.js';
```

<Description markdown={`${cdnJs({ components: ['pagination'] })}`} />
<Description markdown={`${cdnCss()}`} />

### HTML

```html
<cds-pagination page-size="10" total="100">
  <cds-select>
      <cds-select-item value="10">10</cds-select-item>
      <cds-select-item value="20">20</cds-select-item>
      <cds-select-item value="30">30</cds-select-item>
      <cds-select-item value="40">40</cds-select-item>
      <cds-select-item value="50">50</cds-select-item>
  </cds-page-sizes-select>
</cds-pagination>
```

## `<cds-pagination>` attributes, properties and events

`<cds-pagination at-last-page>`) and `false` means not setting the attribute
(e.g. `<cds-pagination>` without `at-last-page` attribute).
