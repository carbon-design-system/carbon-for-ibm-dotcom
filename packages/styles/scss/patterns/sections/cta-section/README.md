### SCSS

#### Usage

Import the package css into the top of your main CSS file.

```css
@import '@carbon/ibmdotcom-styles/scss/patterns/sections/ctasection/ctasection';
```

#### Blocks

| Class              | Description                                          |
| ------------------ | ---------------------------------------------------- |
| `.bx--cta-section` | Highest level class name for the CTA section pattern |
