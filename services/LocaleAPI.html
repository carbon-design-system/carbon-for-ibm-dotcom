(async, static) getLocale() → {Promise.<Locale>}

Gets the user's locale.

Grab the locale from the available information on the page in the following order:

  1. DDO
  2. HTML lang attribute
  3. ipcInfo cookie
  4. Browser (navigator.language)
  5. Default (us-EN)
